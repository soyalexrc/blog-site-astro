# Introduction

Welcome to my first MDX file! This is a **markdown** file that supports **JSX** syntax, which means you can write and render React components directly in your markdown files.

## JSX Example

Here's an example of a JSX component:

```jsx
import React from 'react';

const MyComponent = () => {
  return (
    <div>
      <h1>This is a React component</h1>
      <p>It can be rendered directly in your markdown files using MDX.</p>
    </div>
  );
};

export default MyComponent;
